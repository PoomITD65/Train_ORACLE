# Select statement

### select all

```sql
select * from table;

### select with condition

```sql
select * from table where filename = 'somevalue';
```

### select with join

```sql
select * from table1 t1
join table2 t2 on t1.t1fieldid2 = t2.t2field
join table3 t3 on t1.t1filedid3 = t3.t3field
```
